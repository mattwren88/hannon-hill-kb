Requirements & Supported Platforms for 8.16

Server Requirements (for CMS deployment)

Application server requirements (for the CMS)
  Requirement Notes
CPU
  • 8-core (recommended)
  • Quad core (minimum)
 Cascade can be run in virtualized environments like VMware and Hyper-V.
Memory
  • 8GB RAM (recommended)
  • 6GB RAM (minimum)
 It is recommended to allocate approximately 1/2 of the server’s total memory to Cascade.
Hard Drive Space
  • 25GB (recommended)
  • 15GB (minimum)
